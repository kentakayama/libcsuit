# libcsuit Sample Programs

- Processing & Reporting: Parse and Execute a SUIT Manifest, and Generate a SUIT Report (see [process](https://github.com/kentakayama/libcsuit/tree/master/examples/process))
- Parsing: Parse and Print a SUIT Manifest (see [parser](https://github.com/kentakayama/libcsuit/tree/master/examples/parser))
- Signing Manifest: Sign/MAC a SUIT Manifest (see [sign](https://github.com/kentakayama/libcsuit/tree/master/examples/sign))
- Encrypting & Decrypting a Payload: Encrypt a payload to generate a `SUIT_Encryption_Info` and ciphertext (see [encrypt](https://github.com/kentakayama/libcsuit/tree/master/examples/encrypt))
- Encoding: Encode a (signed) SUIT Manifest (see [encode](https://github.com/kentakayama/libcsuit/tree/master/examples/encode))
